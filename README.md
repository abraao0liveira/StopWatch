## O que é o StopWatch?
A ideia principal so StopWatch é ser um cronômetro com a opção de contagem em segundos, minutos e horas, de forma progressiva e regressiva.

## Pré-requisitos
- .NET

## Como rodar o código?

### [1] Primeiro, rodar os comandos dotnet
```bash
dotnet restore
dotnet build
dotnet clean 
```

### [2] Segundo, rodar a aplicação
```bash
dotnet run
```
